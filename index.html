Visualizador de Ventas Diarias

Versión HTML del tab Ventas diarias. Permite cargar uno o más Excel, aplicar una configuración diaria Categoria ⇄ Grupo para excel + Peso, filtrar por sucursal, año, mes y día de semana, y ver gráfico más tabla sin usar MATLAB.

Esta versión usa librerías desde CDN para leer Excel y dibujar el gráfico. Si luego la quieres 100% offline, se puede hacer una segunda versión empaquetada.
